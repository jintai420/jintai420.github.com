首尾よく３ｐを引いたので、４ｐ切り。カン４ｍが入ればリーチだけど、ピンズが先に埋まれば、３ｍ切って一向聴に戻します。その場合の狙いは、ノベタンの横くっつき。

How to Slove the Problem About VMware Easy Install

Feb 14th, 2014

When I tried to install Ubuntu 12.04 on Vware Fusion, met the problem, which is that VMware said the below text.

vmware easy install

please wait! vmware tools is currently being installed on your system. depending on the version of ubuntu you are installing. you may login in below and use the system during the installation. otherwise, please wait for the graphical environment to lanuch.

Thank you.

Then, I found the way to solve this problem. I write down as memo below.

  1. Login by typing youer username and password.

  2. Restore three files which names are “/etc/issue”, “/etc/rc.local” ,“/etc/init/lightdm.conf”. Just type below commans.


sudo mv /etc/issue.backup /etc/issue
sudo mv /etc/rc.local.backup /etc/rc.local
sudo mv /opt/vmware-tools-installer/lightdm.conf /etc/init
sudo reboot

After reboot, my Ubuntu is availavle.

一貫性の考え方

Feb 14th, 2014

一貫性を保てという言葉、麻雀を打っていると１回は聞いたことがあると思います。一貫性を保っていい目標というのは本来１つだけで、「将来的に自分の順位を上げるために最適な行動をとる」という一点のみです。これ以外は状況に応じてコロコロ変わるものだと思っていい。

局がスタートした最初のテーマは？もちろんあがることですよね。

先制リーチが相手から入った・・・オリますよね。じゃあ、次。

降りてたらたまたま聴牌した・・・リーチ？ダマ？

中級者くらいまでの人がよく迷うシーンの１つとして、降りてたらたまたま聴牌した状況があると思うんです。こういったシーンでは一貫性が求められます。一貫性を保つにはその状況に対する細かい認識が必要です。この場合だと、オリにも２つの種類があることを理解してないといけない。

「聴牌を崩してでも放銃を避ける徹底したオリ」と「放銃する可能性はやや残るけど、聴牌を意識したオリ」。特に尾を引くミスになりがちなのが「オリてるつもりだったのに聴牌したからとりあえずリーチして、相手に打った」系のミスですね。一貫性が崩れて軸がブレるから、次のミスに繋がりやすい。だから、一貫性を保つことでブレを防ぐ。

一貫性を保つもう一つの目的・・・すなわち良好なメンタル状態の維持です。

例えば、マンズの染め手に対して切れないとさっきは判断して止めたマンズ。だけど、今このマンズ１枚を切れば聴牌が取れる・・・そういう時はマンズを切って聴牌をとってはいけないんです。得点を削ってまで取る価値のある聴牌なのかどうなのか？最終的に自分は得するのか？大事なのはそこです。

無駄な頑張りを減らす

Feb 14th, 2014

数日前に検討会をした。そこで気がついたことだけど、初心者になればなるほど、頑張りすぎているなということに気がついた。麻雀があまり上手でない人はどんなにボロボロの手でも、相手から何手遅れていようとも関係なくあがりに向かって突っ込んでいく。そして、放銃する。結果として負ける。

ゲーム性から来るのもだけど、麻雀って粘るとか頑張るという言葉からはほど遠いゲームだと思う。ダメな手のときはひたすらオリて、勝てそうな手はアグレッシブに攻める。一言でいえば、押し引き。押し引きとはどのタイミングで引くかを判断すること。株で言えば、損切りのポイントを見極める行為に近い。そこにあるのは頑張りとは程遠い妥協という言葉だ。麻雀を打っているときは努力すればするほど、泥沼にハマっていく。（もちろん、打っていないときは検討や反省する努力は必要だが・・・）。

自分が下手なときも例にもれず、よく突っ張って放銃していたように思う。そして打つ、負ける。挙句の果てにふてくされる。そういうことを繰り返して、だんだん引き際のコツを掴んでいく。そういう体に覚えこませる作業みたいな部分を見てると、麻雀ってスポーツに似てるなと思う。

読みのロジック

Feb 14th, 2014

4sに続いて赤の5s切ってリーチ。両方手出し。

この手出しを見て真っ先に疑うのべき待ちは６－９ｓの１点のみ。

赤払ってまでリーチして、なおかつ良さそうに思える待ち。基本的に待ちは端の方がよい待ちになりやすい。３−６ｓより６−９ｓ、そしてドラは７ｓ・・・。だから６−９ｓはクサイ。

こんな所でしょうか。

「読む」とは、相手のことを考えることです。相手の視点に立ってものが見えるとき、はじめて「読む」という行為を行っていることになる。

麻雀における読みの位置づけは結構低いです。少なくとも最重要事項には決して挙がらない。何故かと言うと、仮に相手の待ちを読み切っても当たり牌を打たなければならないシーンだってあるし、ツモられれば待ちを当てようが当てまいが関係がない。自分の目に移る範囲の情報が少し増える程度のものなんです。

では何故読みが必要かというと、相手のアクションを予想することで、こちらもそれに沿った対応を取れるからです。要は相手のやりたいこと、その局のテーマを予想する。それに沿った対応を行う・・・。

じゃあ、どこを見ればいいんだよ！ってなるんですけど、主に見る箇所は３つあります。１つは役と手。２つめは状況に沿った各プレイヤーの狙い。３つめはキャラ読み。これはまた別の記事で書きます。

あと余談ですけど、個人的に手牌読みはあまり推奨してません。枚数を数えたり、形をイメージしたりするのはやってみるとわかるけど、かなり疲れます。３家の手出しツモ切りと形と枚数をすべてカウントするのは実質ほぼ不可能ですから。

リーチをかける意義

Feb 14th, 2014

リーチには２つの種類があります。

  1. あがるためのリーチ
  2. 防御のためのリーチ

正直、２つに分ける意味はあまりないですｗ根っこにある考えは同じものなので。ただ、こういうふうにカテゴリに分けると少し見やすくなると思ったので、ここではあえて２つに分けてみました。

リーチをかける意義ってなんでしょう？ 多くのプレイヤーは「聴牌したからリーチ！」「あがりたいからリーチ！」が主な動機なのではないかと思います。僕の考えは少し違って、「あがれそうだからリーチ」「リーチしないとマズイからリーチ」を基準に考えています。さっき上で説明した２点ですね。つまり、自分にとって得になるからリーチなんです。

あがるためのリーチ・・・リーチすれば出やすい待ちで釣るため。ツモって点数を叩き出すため。

防御のためのリーチ・・・リーチして相手を牽制する。先にあがって相手の大きい手を潰す。

簡単にいえば、自分の中にリーチかける動機があって、それと状況が噛み合ったときにリーチなんです。それは「あがり牌が少ないから、その虚を突くため」であっても良い。ただ、リーチを打つということはゼンツするリスクを犯しているということです。このリスクを加味することがポイントです。簡単に書くと・・・。

リターン＞リスクならリーチ

リスク＞リターンならダマ

上に書いたように、とても単純な話なんです。 このリスクとかリターンを加味した判断基準を自分の中に作っていく、洗練させていくことが上達につながるんです。

Spaghetti Aglio Olio e Peperoncino

↓レシピはこちら

ゆずペペロンチーノ （2人分）

・スパゲッティ（乾、1.6mm）160g

・にんにく5かけ

・顆粒チキンスープの素小さじ2

・柚子こしょう（市販）小さじ1

・クレソン2本

1.なべに湯を沸かしスパゲッティをゆでる。

2.にんにくは木べらをのせてたたきつぶす。

3.フライパンにオリーブ油カップ1/4とにんにくを入れて 弱火にかける。にんにくの角が薄く色づくまで、3分間程 じっくりいため、いったん火を止めておく。

4.スパゲッティのゆで湯からカップ1/2を取り分け、チキン スープの素、柚子こしょうを混ぜておく。

5.スパゲッティの湯をきり、3のフライパンに加えて強火に かける。4も加えてしっかりと混ぜ、皿に盛りクレソンを飾る。
